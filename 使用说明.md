# 自动视频脚本工作台 · 使用说明与配置指导

本软件把网址或长文本（小说等）整理成多人物对话短剧，可按集生成画面、分角色配音、字幕，并合成最终视频。下面按「打包 → 安装 → 配置 → 使用 → 排错」顺序说明。

---

## 一、打包成免安装 exe（开发者/首次准备时做一次）

> 说明：Windows 的 `.exe` 必须在 **Windows 电脑** 上打包；macOS 同理在 Mac 上打包。打好后的程序可拷给非技术用户双击运行。

### Windows
1. 安装 **Python 3.10 或更高版本**（python.org，安装时务必勾选 **Add Python to PATH**）。
2. 把本项目文件夹整个拷到电脑上。
3. 双击运行 **`build_windows.bat`**。脚本会自动装依赖并打包。
4. 完成后，可执行文件在 **`dist\AutoVideoStudio.exe`**。

### macOS / Linux
```bash
bash build_macos.sh
# 产物在 dist/ 目录
```

### ffmpeg（视频合成必需）
程序用 ffmpeg 做拼接/配音/字幕合成。二选一：
- **推荐**：下载 `ffmpeg.exe` 和 `ffprobe.exe`，放到 `dist\` 目录里（和 exe 同级），即可随程序一起分发；
- 或：在系统里安装 ffmpeg 并加入 PATH（macOS：`brew install ffmpeg`）。

---

## 二、安装与首次运行（最终用户）

1. 拿到 `AutoVideoStudio.exe`（建议连同同目录的 `ffmpeg.exe`/`ffprobe.exe` 一起）。
2. 双击运行。
3. Windows 10/11 一般自带 **WebView2 运行时**；若提示缺少，去微软官网搜索 “WebView2 Runtime” 安装一次即可。

程序运行后会在 **exe 所在目录** 生成 `outputs\`（成片与脚本）、`config.json`（你的配置）、`ui_config.json`（主题等）。

---

## 三、你需要做的配置（点右上角 ⚙ 设置）

设置面板分三块。**不填任何 Key 也能用本地规则跑通流程**（仅用于预览结构，不出真实视频/配音）。要真正出片，需要下面的 Key。

### 1) 文本模型（可选，让故事/剧本更好）
- OpenAI Chat Completions 兼容接口。
- 填：接口地址、API Key、模型名。
- 不填则用本地规则生成（只能预览结构，无法写出真正的人物对白）。

### 2) 视频生成 · Grok / xAI（出画面必需）
- 获取 Key：登录 **https://console.x.ai** → API Keys 创建。
- 在设置里填 **xAI API Key**；可选画面比例（竖屏 9:16 等）、分辨率、一致性（建议「强一致+帧衔接」）。

### 3) 配音 · Gemini / Google（人物说话必需）
- 获取 Key：登录 **https://aistudio.google.com** → 创建 API Key。
- 在设置里填 **Gemini API Key**；可选默认音色（每个角色还会自动分配固定音色）。

> 填好点「保存」即时生效，无需重启。顶部状态标签会显示是否已连接文本模型。
>
> ⚠️ Grok 视频与 Gemini 配音是**按量付费**服务，几十集会有相应费用，请先用 1 集试跑确认效果与花费。

## 四、使用流程

1. **选输入**：粘贴长文本/小说，或切到「网址」填链接（可先「抓取网址正文预览」确认内容）。
2. **1·编排故事**：生成故事梗概、**角色表**、**场景表**、风格。
3. **2·生成多个脚本**：拆成多集对白剧本（场景 + 谁说什么）。生成后会显示**调用量与耗时预估**。
4. 选中某集 → 右侧详情看对白。可：
   - **预览参考图**：先看角色/场景定妆图，确认画面基准；
   - **编辑本集**：改标题/对白（保存后该集标记为需重做）；
   - **3·制作选中集**：生成这一集的成片（画面+配音+字幕）；
   - **重做本集**：单独重渲染这一集。
5. **制作全部集**：一键批量。已完成的自动跳过，失败的继续下一集，**关掉重开也能从断点续跑**。
6. **打开输出文件夹**：查看成片。

## 五、输出位置

```
outputs/
  时间戳_故事标题/
    story/        故事、角色表、场景表
    scripts/      每集剧本（.json/.md）
    assets/
      characters/ 各角色定妆参考图（跨集复用，保证人物一致）
      scenes/     各场景参考图
    videos/       每集成片 .mp4 与字幕 .srt
    jobs.json     每集制作状态（用于断点续跑）
```

## 六、常见问题

- **双击闪退/白屏**：多为缺 WebView2 运行时，装一次即可；仍不行可让开发者把 `videostudio.spec` 里 `console=False` 改成 `True` 重新打包，从命令行运行看报错。
- **提示未检测到 ffmpeg**：把 `ffmpeg.exe`/`ffprobe.exe` 放到 exe 同目录，或装好并加入 PATH。
- **字幕中文显示成方框**：系统缺中文字体；装一个中文字体（如思源黑体 / Noto Sans CJK）即可；程序会自动退化为软字幕并始终保留 `.srt`。
- **某集失败**：批量会跳过它继续；修好 Key/网络后再点「制作全部集」会自动续跑失败/未完成的集。
- **人物在不同集长得不一样**：确保用「强一致+帧衔接」，且角色表里给了清晰的外貌描述；参考图只在第一次生成、之后全程复用。
- **首次接口字段报错**：不同厂商接口可能有细微差异，把报错发给开发者，改 `providers.py` 一处即可校准。
